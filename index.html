Modular packages for cross platform developement
Simple ValidationThe root package is@xpfw/validate
It's kind of like JSON-Schema but includes relationship and permission options similar to Silverstripes DbObject.
Easy Forms@xpfw/form lets you display and customize forms based on @xpfw/validate definitions.
This is comparable to react-jsonschema-form or Meteor's autoform,  but it also supports React Native out of the box.
Admin / CMS Ready@xpfw/ui like Meteor abstracts simple CRUD and paginated Lists for collections and comes with readily usable React UI.
Think of react-admin written for @xpfw/validate definitions.
Backend Independent@xpfw/ui has been written backend independently.
Simply implement IBackendClient or use existing ones like feathers.
All driven by one definition
import { FieldType, IField, IForm, RequiredType, ValidationRegistry } from "@xpfw/validate"
const RecipeModel: IForm = {
  model: "recipeModel",
  collection: "recipes",
  sections: [{fields: [{
    mapTo: "name",
    type: FieldType.Text,
    validate: {required: {type: RequiredType.Always}}
  }, {
    mapTo: "createdBy",
    type: FieldType.Text,
    validate: {required: {type: RequiredType.Always}}
  }, {
    mapTo: "createdAt",
    type: FieldType.Date
  }]}]
}
ValidationRegistry.registerForm(RecipeModel)
