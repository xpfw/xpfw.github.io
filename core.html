Easy DefinitionsSimple but flexible Form definitions allow all higher level packages to do as much as possible automatically.
Relationships@xpfw/ui shows how easy Relationships are with a simple addition to a definition.
Permission SupportThere's an optional permission system explained in the docs, thats helpful if used in a backend.
Roll your own stack@xpfw/validateexports validateForm  and validatePermission to make it easy to intergrate into your own backend. Need to connect the UI as well? Check out IBackendClient.
Readily Usable Stacks
If you don't wish to adapt xpfw to your needs you can use the following backend suggestions, which work out of the box.
FeathersHave a full stack ready and waiting by using @xpfw/feathers hooks in a feathers application. Connect to it via @xpfw/ui-feathers. A full example can be found in the docs.
NeDBWant an offline app? Use @xpfw/ui-nedb to persist data in localStorage! Read the example in the documentation if you're curious!
